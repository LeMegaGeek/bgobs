# BGOBS 0.3.6

- Distinguish an open USB interface from a completed CaCam handshake.
- Warn after five seconds without USB data and suggest unlocking the phone and
  keeping CaCam in the foreground.
- Complement the CaCam 0.9.8 foreground-start fix for phones launched by
  Android Open Accessory.

Tested with CaCam 0.9.8.
