# BGOBS 0.3.11

- Rename the CaCam source and expose direct USB and ADB connection modes.
- Add ADB mode, which starts CaCam, creates a local port forward, and reads HTTP
  snapshots without an OBS browser source.
- Add Rotten, Low, Standard, HD, and UHD qualities with a separate BGOBS
  optimization setting.
- Include Android platform-tools in the Windows package when they are available
  at build time.
