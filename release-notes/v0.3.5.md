# BGOBS 0.3.5

- Reduit la latence de **CaCam USB** avec une sortie OBS non bufferisee et le rejet des frames USB obsoletes.
- Stabilise la mesure de latence sur les longues sessions en compensant la derive d'horloge Android/PC.
- Ameliore la detection et la negociation Android Open Accessory, notamment sur les telephones Xiaomi.
- Conserve la connexion lors des changements de scene et reprend automatiquement apres une deconnexion.
- Ajoute `install-portable.bat` pour OBS PortableApps et reconnait le lanceur `OBSPortable.exe`.
- Inclut `libusb-1.0.dll` dans le paquet Windows pour rendre la source **CaCam USB** disponible.

Version Android associee : CaCam 0.9.6.
