# BGOBS 0.3.5

- Reduce CaCam USB latency with unbuffered OBS output and stale-frame drops.
- Stabilize latency estimates over long sessions by compensating for Android
  and host clock drift.
- Improve Android Open Accessory detection and negotiation, including Xiaomi
  phones.
- Preserve the connection across scene changes and resume after disconnects.
- Add `install-portable.bat` for OBS PortableApps and recognize
  `OBSPortable.exe`.
- Include `libusb-1.0.dll` in the Windows package.

Tested with CaCam 0.9.6.
