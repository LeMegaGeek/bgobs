How to Install BGOBS on Windows
=============================================================

PortableApps OBS Studio
-----------------------

1. Close OBS Studio.

2. Run:

     install-portable.bat

   When prompted, enter the folder that contains OBSPortable.exe (or the
   legacy OBS-StudioPortable.exe launcher). The script copies all DLL and data
   files automatically.

3. Restart OBS Studio and add the source:

     CaCam USB

   libusb-1.0.dll must remain beside bgobs.dll for this source to be available.


Manual PortableApps installation
--------------------------------

1. Remove older installations before copying the new one.

   Run:

     remove-old-installation.bat

   When prompted, enter the folder that contains OBSPortable.exe.

   Example:

     D:\PortableApps\OBS-StudioPortable\

   This removes both older obs-backgroundremoval files and previous BGOBS files.

2. Find the folder that contains OBSPortable.exe.

   Example:

     D:\PortableApps\OBS-StudioPortable\

3. Copy the DLL files from:

     bgobs\bin\64bit\

   into:

     D:\PortableApps\OBS-StudioPortable\App\obs-studio\obs-plugins\64bit\

4. Copy the contents of:

     bgobs\data\

   into both of these folders:

     D:\PortableApps\OBS-StudioPortable\App\obs-studio\data\obs-plugins\bgobs\
     D:\PortableApps\OBS-StudioPortable\Data\obs-plugins\bgobs\

5. The final layout should include:

     D:\PortableApps\OBS-StudioPortable\App\obs-studio\obs-plugins\64bit\bgobs.dll
     D:\PortableApps\OBS-StudioPortable\App\obs-studio\obs-plugins\64bit\libusb-1.0.dll
     D:\PortableApps\OBS-StudioPortable\App\obs-studio\obs-plugins\64bit\onnxruntime.dll
     D:\PortableApps\OBS-StudioPortable\App\obs-studio\obs-plugins\64bit\onnxruntime_providers_shared.dll
     D:\PortableApps\OBS-StudioPortable\App\obs-studio\data\obs-plugins\bgobs\models\
     D:\PortableApps\OBS-StudioPortable\Data\obs-plugins\bgobs\models\

6. Restart OBS Studio and check the source list for:

     CaCam USB

   The video source filter list should also contain:

     Make me look good / Rend-moi beau gosse
     Low-Light Enhancement

7. If OBS still reports version 1.3.x, open the latest OBS log under:

     D:\PortableApps\OBS-StudioPortable\Data\config\obs-studio\logs\

   Search for:

     bgobs

   Delete the old path shown in the log, then copy the new DLL and data again.


Standard OBS Studio
-------------------

Copy the `bgobs` folder to your OBS Studio plugins directory.

The supported default path is:

  C:\ProgramData\obs-studio\plugins\

If `C:\ProgramData\obs-studio` or `C:\ProgramData\obs-studio\plugins` do not
exist, create them manually first.

Do not install this plugin under `C:\Program Files\obs-studio`.
