How to Install OBS Background Removal Plugin on Windows
=============================================================

PortableApps OBS Studio
-----------------------

1. Close OBS Studio.

2. Remove older installations before copying the new one.

   Run:

     remove-old-installation.bat

   When prompted, enter the folder that contains OBS-StudioPortable.exe.

   Example:

     D:\PortableApps\OBS-StudioPortable\

   This matters if OBS still loads an old 1.3.x DLL from a legacy
   obs-plugins\64bit folder.

3. Find the folder that contains OBS-StudioPortable.exe.

   Example:

     D:\PortableApps\OBS-StudioPortable\

4. Open or create this folder:

     D:\PortableApps\OBS-StudioPortable\Data\obs-studio\plugins\

5. Copy the `obs-backgroundremoval` folder from this ZIP into that `plugins`
   folder.

6. The final layout should include:

     D:\PortableApps\OBS-StudioPortable\Data\obs-studio\plugins\obs-backgroundremoval\bin\64bit\obs-backgroundremoval.dll
     D:\PortableApps\OBS-StudioPortable\Data\obs-studio\plugins\obs-backgroundremoval\data\models\
     D:\PortableApps\OBS-StudioPortable\Data\obs-studio\plugins\obs-backgroundremoval\data\locale\
     D:\PortableApps\OBS-StudioPortable\Data\obs-studio\plugins\obs-backgroundremoval\data\effects\

7. Restart OBS Studio and check the video source filter list for:

     Background Removal
     Low-Light Enhancement

8. If OBS still reports version 1.3.x, open the latest OBS log under:

     D:\PortableApps\OBS-StudioPortable\Data\obs-studio\logs\

   Search for:

     obs-backgroundremoval

   Delete the old path shown in the log, then copy the new folder again.


Standard OBS Studio
-------------------

Copy the `obs-backgroundremoval` folder to your OBS Studio plugins directory.

The supported default path is:

  C:\ProgramData\obs-studio\plugins\

If `C:\ProgramData\obs-studio` or `C:\ProgramData\obs-studio\plugins` do not
exist, create them manually first.

Do not install this plugin under `C:\Program Files\obs-studio`.
